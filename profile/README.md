## What is Docryn?

Docryn is an open-source platform for building modern business applications using a **Document-Driven Architecture**.

Instead of manually building user interfaces, your backend describes the application as structured documents.

The Docryn Runtime interprets these documents and renders a complete, interactive application.

```text
Application
      │
      ▼
 Document
      │
      ▼
 Protocol
      │
      ▼
 Runtime
      │
      ▼
 User Interface
```

## Vision

We believe application development should be based on **describing interfaces**, not manually building them.

The server defines the application.

The runtime renders it.
